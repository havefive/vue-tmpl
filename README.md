# vue-tmpl
vue-cli + vue-router + element-ui + axios,enjoy Vue!

## Steps:

### Install vue-cli
```
$ npm install -g vue-cli
```

### Init one project

```
vue init webpack vue-tmpl

cd vue-tmpl

npm install
```

### Add modules(router:vue-router,UI:element-ui,Ajax/Data:axios)

```
npm install vue-router element-ui axios --save
```

### Start dev

```
npm run start
```
### Build product
```
npm run start
```
